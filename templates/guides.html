学習の進め方（英検5級・4級・3級）

Eiken Practice で、受験級ごとに「どのメニューを・どの順で」進めるかをまとめたガイドです。 公式の本試そのものではなく、過去問形式の練習をブラウザで解くためのサイトです。

問題演習はアカウント登録後に利用できます。まずは下の級別セクションで、自分の級の進め方を確認してください。

英検5級の進め方

入門レベル向けです。長文読解・ライティングはありません。 語彙・文法、会話補充、語順、リスニングの基礎固めが中心になります。

このサイトの主なメニュー:

文法・語彙 会話補充 語順選択 リスニング第1〜3部 ランダム10問 模擬試験

おすすめの学習順

  1. 文法・語彙で基礎を確認する
  2. 会話補充・語順選択で短文系に慣れる
  3. リスニングを第1部 → 第2部 → 第3部の順で進める
  4. 慣れてきたらランダム10問で弱点を洗い出す
  5. 週末などに模擬試験で通し練習する

ランダム10問は文法・会話・語順・リスニングから出題されます。 長文は含まれません（5級に長文メニュー自体がありません）。

英検4級の進め方

初級〜標準向けです。5級に加えて長文読解があります。 形式が増えるので、最初から全部をこなそうとせず、カテゴリを分けて回すのがおすすめです。

このサイトの主なメニュー:

文法・語彙 会話補充 語順選択 長文読解 リスニング ランダム10問 模擬試験

おすすめの学習順

  1. 文法・語彙で正答率の底上げをする
  2. 語順選択・会話補充で短文系を固める
  3. リスニングを部ごとに少しずつ進める
  4. 長文読解は時間を取るので、別枠で週に数回
  5. ランダム10問 → 模擬試験、の順で総合力を確認する

ランダム10問には長文は含まれません。長文は専用メニューと模擬試験で練習してください。 模擬試験は本番形式の総合問題（長文込み）です。基本問題の各カテゴリで取り組み率が上がると解放されます。

英検3級の進め方

中級向けです。ライティングがあり、読解・リスニングも実践寄りの内容になります。 選択問題とライティングは別メニューなので、役割を分けて使うと迷いが減ります。

このサイトの主なメニュー:

文法・語彙 会話補充 ライティング 長文読解 リスニング ランダム10問 模擬試験

おすすめの学習順

  1. 文法・語彙と会話補充で、選択問題のベースを作る
  2. 読解とリスニングを部ごとに継続する
  3. ライティングは別メニューで、模範解答と比較しながら練習する
  4. ランダム10問で短時間の総合確認をする
  5. 模擬試験で選択式・読解・リスニングの通し練習をする

ランダム10問には長文・ライティングは含まれません。 模擬試験もライティングは別メニュー扱いです。ライティングは専用メニューで提出し、解説・模範解答を確認してください。

共通の使い方のコツ

  • 毎日少しでよい — デイリーミッションで目標問数（例: 3・5・10問）を決めて続ける
  • 間違えた問題は解説を読む — 正答率より「見直せたか」を優先する
  • ランダムと模擬は後半 — 各形式をある程度やってから解いた方が効果的
  • スマホでもOK — ブラウザで動作します。リスニングはイヤホンがあると集中しやすいです

フィードバックの送り方

問題の誤字・音声の不具合・使いづらい点・ほしい機能などがあれば、アプリ改善のためにフィードバックを送れます。 ログイン後、画面上部メニューの「フィードバック」から利用できます。

選べる種類:

バグ報告 機能要望 改善提案 その他
  • タイトルと内容を書いて送信します（内容は最大5,000文字）
  • 返信が必要な場合は、メールアドレス欄に連絡先を入れてください（任意）
  • どの級・どの問題形式で起きたか書いてもらえると、調査しやすいです

フィードバックはログインが必要です。まだアカウントがない場合は、先に無料登録してからお送りください。

よくある質問

公式の過去問そのものですか？

公式試験そのものではありません。過去問形式の練習問題を、ブラウザ上で解けるようにした学習サイトです。

どの級から始めればいいですか？

受験予定の級を選んでください。級ごとにメニュー構成が異なります。このページの級別セクションで進め方を確認できます。

リスニングの音声が鳴りません

端末の消音、ブラウザの自動再生制限、イヤホン接続を確認してください。ページを再読み込みしてから再生ボタンを押すと改善することがあります。解消しない場合は、ログイン後のフィードバックから状況を教えてください。

毎日どれくらい解けばいいですか？

まずは1日数問で構いません。デイリーミッションで目標を決め、続けることが大事です。

不具合や要望はどこから伝えられますか？

ログイン後、画面上部メニューの「フィードバック」から送れます。バグ報告・機能要望・改善提案・その他を選べます。
